Ferramentas para Desenvolvimento


Para que um profissional WebDesigner possa desempenhar bem suas atividades, além de estudar e conhecer bem as linguagens e técnicas de programação, é essencial que ele utilize boas ferramentas para desenvolvimento web.

Essas ferramentas irão ajudar o WebDesigner a ser mais ágil, eficiente e a produzir conteúdos com alta qualidade. Em um mercado extremamente concorrido como esse, a produtividade e a qualidade devem ser fatores chave e é por isso que a utilização dessas ferramentas de desenvolvimento são tão importantes.

É importante lembrar que não adiantar usar uma ferramenta para edição HTML se você não conhece nada de HTML. Você até conseguirá criar uma página sem muitas dificuldades, mas certamente terá dificuldades para resolver problemas e criar soluções mais customizadas. Um verdadeiro profissional WebDesiger deve conhecer a linguagem e só então se utilizar essas ferramentas de desenvolvimento Web para agilizar o trabalho, e isso vale para todas as linguagens.

Nessa página vamos citar as melhores e mais utilizadas ferramentas do mercado, tanto para edição gráfica quanto para edição de código. Cada software tem suas características e cabe a cada profissional testar e escolher a que ele melhor se adapta.

IDE


Como as pessoas começam como usuários de computador onde o normal é usar um Microsoft Word por exemplo, e aquilo faz tudo o que ele precisa em termos de edição de texto, quando ela vai programar e mandam ela instalar o Visual Studio, o Eclipse, o Android Studio ou algum outro IDE ela acha que aquilo é a linguagem de programação.

O IDE (Integreted Develpment Environment) é o ambiente de desenvolvimento integrado (note que é masculino), ou seja, é o programa que integra as várias ferramentas necessárias para o desenvolvimento de softwares ajudando todo o processo ficar mais fácil.

Sua principal função é ajudar o programador editar o código que será usado para criar seu programa.

Outra coisa muito importante que todos IDEs fazem é chamar o compilador, ou interpretador, dentro de certos parâmetros para gerar e/ou executar o programa criado se ele não contiver erros.

Ele também ajuda:
- depurar programas durante a execução com várias facilidades;
- gerenciar os projetos e suas configurações
- executar testes, montar o deploy ou até mesmo fazê-lo;
- fazer análise estática;
- controlar de versão;
- acessar banco de dados;
- gerar códigos automaticamente para certos padrões de codificação, como telas e relatórios;
- dar acesso fácil à documentação;
- diversas ajudas durante o processo de edição do código.

Isso pode variar de IDE para IDE e como ele está configurado. Como o nome diz, estas ferramentas costumam ser integradas. Claro que algumas estão lá por padrão, mas muitas delas dependem de um programa externo para executar a tarefa. O próprio compilador da linguagem normalmente é algo externo.
Um IDE pode suportar várias linguagens (ter ferramentas específicas para apoiar o uso de determinada linguagem).


Ferramentas de Edição Gráfica
Dentre as atividade de um WebDesigner, estão as manipulações gráficas. O profissional regularmente terá que criar e manipular imagens e fotos, criar banners, logos, layouts e etc. Para isso as seguintes ferramentas são frequentemente utilizadas:

Adobe Photoshop
O Photoshop é sem dúvidas o software para edição e tratamento de imagens mais utilizado. Suas funcionalidades são otimizadas para edição e tratamento de imagens digitais e trabalhos de pré-impressão, ideal para tratamento de fotografias e criação de peças de design cheias de efeitos.
O domínio do Photoshop é essencial para todo profissional Webdesigner de ponta pois qualquer trabalho gráfico para um Website irá em algum momento ser feito nele.

Adobe Fireworks
O Fireworks foi feito para WebDesigners. A ferramenta possui seus recursos otimizados para o desenvolvimento de Layouts Web. Possui boa compreensão de imagens, PNGs transparentes, imagens fatiadas, organização por páginas e etc.
Existe uma certa concorrência e discussões entre o uso do Adobe Photoshop e/ou do Fireworks. Na realidade, não há nada que o Fireworks possa fazer que o Photoshop não possa, mas com o Fireworks algumas tarefas corriqueiras do web designer podem ser feitas com mais facilidade, permitindo projetar websites de uma forma mais rápida e objetiva.

Corel Draw
Ferramentas Desenvolvimento WebO Corel é um velho conhecido. É uma ferramenta muito utilizada para criação de imagens vetoriais.
Apesar de ter perdido muito mercado para o Illustrator, ainda é muito usado no Brasil, principalmentes por iniciantes na área devido à sua facilidade de uso.

Adobe Illustrator
O Illustrator é atualmente o principal software para criação de imagens vetorias. Ele possibilita e facilita a criação de ilustrações, logotipos, folhetos, livros, layouts, infográticos, cartões e etc.
É importante citar que os 3 software da Adobe (Photoshop, Fireworks e Illustrator) se integram muito bem e por isso é comum os profissionais utilizarem todos os softwares em conjunto.

Ferramentas de Edição de Código
No arsenal de ferramentas para desenvolvimento Web de um profissional, os softwares de edição de código tem um lugar muito especial. Eles são muito úteis para agilizar a criação de páginas com HTML, arquivos CSS e códigos em JavaScripts, PHP e etc. Vamos listar aqui os principais aplicativos utilizados no mercado

Adobe Dreamweaver
Softwares para Desenvolvimento WebEsta é uma das principais ferramentas de desenvolvimento de aplicações Web. O programa permite a criação e edição de páginas sem a necessidade de se lidar diretamente no nível de código, permitindo que desenvolvedores de qualquer nível possam criam websites com um excelente visual e ótimas funcionalidades rapidamente.
Inicialmente o Dreamweaver era apenas um editor HTML, porém após diversas evoluções hoje ele é uma solução completa para desenvolvedores Web suportando além do HTML, CSS, JavaScript, PHP, ASP, JSP, Ajax e muitas outras linguagens e tecnologias.

WebNotepad++
O Notepad++ (plus plus) é um excelente editor de códigos, ele é bem mais simples, limpo e genérico, não é voltado apenas para linguagens utilizadas no desenvolvimento Web. Outra ponto importante é que o software é gratuito.
Muita gente prefere o Notepad++ justamente por ele ser mais simples e por ser um editor que efetivamente te ajuda a desenvolver o código, e não que faz o código para você, como é o caso do Dreamweaver.
Conheça também os melhores editores de código para PHP e para HTML e CSS.
